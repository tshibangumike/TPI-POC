Release notes for Kendo UI Core v2016.2.607

Components

What's new

  • Group layers by interaction state in theme PSDs

What's fixed

  • TypeScript: Missing footerAttributes in GridColumn
  • Navigator does not allow selection of the last date
  • TypeScript definitions yield errors upon compilation in VS2015

AngularJS

What's new

  • Support k-data-source configuration attribute

Calendar

What's fixed

  • setOptions does not allow clearing the dates option

ColorPicker

What's fixed

  • ColorPicker should not open when disabled via API method or label click

ComboBox

What's fixed

  • Remove aria-readonly attribute from role="listbox"/role="combobox"

DataSource

What's new

  • Add Symbol.iterator support to ObservableArray to allow ES6 for-of, etc.

What's fixed

  • Destroyed records are persisted when DataSource is re-read
  • Data Source should mark offline items in __create__ state as dirty when they are read from the offline storage in order to correctly sync them when it becomes online

DatePicker

What's fixed

  • Remove aria-readonly attribute from role="listbox"/role="combobox"

DateTimePicker

What's fixed

  • Remove aria-readonly attribute from role="listbox"/role="combobox"

Draggable

What's new

  • Support autoscroll when custom hint is used

Drawing

What's fixed

  • An error is thrown by the canvas surface for elements without bounding box
  • kendo.drawing.pdf.saveAs does not work in Firefox
  • repeatHeaders:true duplicates Grid headers if Grid scrolling is disabled

DropDownList

What's fixed

  • Remove aria-readonly attribute from role="listbox"/role="combobox"
  • Home/End navigation does not trigger change when popup is closed
  • Widget makes two requests when cascaded

Excel Export

What's fixed

  • Grid lines are hidden by default in exported Excel file
  • Rowspan configuration may cause some cells to disappear
  • Misplaced headers when exporting to excel

Globalization

What's fixed

  • kendo.parseDate does not parse ko-KR dates correctly
  • kendo.toString puts group separator in the wrong place when value is negative

MultiSelect

What's fixed

  • Remove aria-readonly attribute from role="listbox"/role="combobox"

Notification

What's fixed

  • Notification popups that are currently being hidden should not be returned by getNotifications()

PDF Export

What's fixed

  • Error when exporting PDF content with only 'margin' specified

PivotGrid

What's new

  • Support k-data-source configuration attribute
  • Enhance icon sizes in Material themes

TabStrip

What's fixed

  • TabStrip border is not correctly styled in the Nova theme

TimePicker

What's fixed

  • Remove aria-readonly attribute from role="listbox"/role="combobox"
